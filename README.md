## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`

## Run with Docker

**Prerequisites:** Docker and Docker Compose

1. Build and start the container:
   `docker-compose up --build`

The application will be available at http://localhost:8080.
